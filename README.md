# OBMC Intramurals Registration

## Setup

```bash
pip install -r requirements.txt
```

## Running

```bash
python app.py
```

Server runs on `http://localhost:5000`

Open the HTML files in your browser (index.html, team-checker.html, players.html)

## Files

- `app.py` - Flask backend with team assignment logic
- `index.html` - Registration page
- `team-checker.html` - Team eligibility checker
- `players.html` - Player list and search
- `players_data.json` - Stored player data
- `run.sh` / `run.bat` - Startup scripts for Mac/Linux and Windows
